Міні-проект з Основ програмування
Команда: Усенко Вікторія та Костенко Дарина
Історичні дані про акції ExxonMobil з біржовим тикером XOM були отримані за допомогою бібліотеки yfinance. Для аналізу було обрано період із 1 січня 2023 року до 1 липня 2026 року.
Дані містили інформацію про ціну відкриття, ціну закриття, максимальну та мінімальну ціну акції, а також обсяг торгів.
Для аналізу ціни закриття були розраховані дві ковзні середні:
1.50-денна ковзна середня, яка відображає короткострокову тенденцію
2.200-денна ковзна середня, яка відображає довгострокову тенденцію.
Для створення торгових сигналів використовувалася стратегія перетину ковзних середніх:
1. Сигнал Buy створювався, коли 50-денна ковзна середня перетинала 200-денну знизу вгору
2. Сигнал Sell створювався, коли 50-денна ковзна середня перетинала 200-денну зверху вниз
3. В усі інші дні використовувався сигнал Hold.
Для кожної завершеної операції розраховувався прибуток або збиток.
У проєкті використовувалися такі бібліотеки:
1. yfinance - для отримання історичних біржових даних
2. pandas - для роботи з таблицями, очищення даних і розрахунку ковзних середніх
3. matplotlib - для побудови графіків ціни акції та ковзних середніх.
